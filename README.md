This is the source code for [acherniske.github.io](https://acherniske.github.io), the personal portfolio of Aiden Cherniske — 
hardware engineer, embedded systems developer, and digital design enthusiast.

## Features
- **PCB-inspired hero section** with interactive SVG background
- **About**: Education, experience, and a card-based skills grid with technology logos
- **Projects**: Featured and selected hardware/software projects with sticky action links
- **Featured In**: Press and articles where Aiden is mentioned, with PDF and web links
- **Contact**: Modern, accessible contact section
- **Responsive design**: Looks great on all devices
- **Custom branding**: Favicon and footer wordmark

## Tech Stack
- **React 18** + **TypeScript**
- **Vite** for fast development
- **Tailwind CSS v4** for styling
- **Lucide React** for icons
- **SVG assets** for logos and branding

## Project Structure
```
├── public/

│   └── ...
├── src/

│   ├── components/     # React components

```
